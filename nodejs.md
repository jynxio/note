# node.js



## 安装

1. 下载：https://nodejs.org ，推荐下载最新版的 LTS ，推荐理由见 npm 的《检查更新》
2. 安装



## 更新

​		window 系统必须通过重新安装来控制 Node.js 的版本。



## 检查版本

```
node -v
```



# cmd

```
md filename    新建文件夹

cd ..          返回上一级目录
cd \           返回至根目录

dir            打印当前文件夹下的文件的目录

cls            清屏

exit           退出

pwd            输出当前目录的地址
```
